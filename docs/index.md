## PHP Collections Library

#### Installation

You can install library with Composer:
```shell
composer require squirrafe/collections
```

#### Version support

| PHP Version | Is supported? | Latest library version with support                                            |
|-------------|---------------|--------------------------------------------------------------------------------|
